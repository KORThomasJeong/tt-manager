// WHAT IT DOES
FEATURES
▸ LIST
세션 목록
실행 중인 tmux 세션 이름과 윈도우 수를 박스 UI로 표시합니다.
▸ ATTACH
자동 attach / switch
tmux 외부면 attach-session, 내부면 switch-client를 자동으로 선택합니다.
▸ NEW
세션 생성
이름 중복 시 attach 여부를 묻습니다. tmux 내부에서 생성하면 즉시 전환됩니다.
▸ KILL
세션 종료
삭제 전 확인 프롬프트를 표시해 실수로 인한 세션 삭제를 방지합니다.
▸ RENAME
이름 변경
이름 중복 체크 후 변경합니다. 실패 시 액션 메뉴로 돌아갑니다.
▸ MOUSE
마우스 자동 활성화
~/.tmux.conf에 set -g mouse on을 자동으로 추가합니다.
// SETUP
INSTALLATION
01
저장소 클론
$ git clone https://github.com/yourname/tt-manager.git $ cd tt-manager
02
실행 권한 부여
$ chmod +x tt.sh
03
전역 설치 (선택)
$ sudo ln -s "$(pwd)/tt.sh" /usr/local/bin/tt
04
실행
$ ./tt.sh # 또는 전역 설치 후 $ tt
// CONTROLS
KEY REFERENCE
MAIN SCREEN
1–9 세션 선택
n 새 세션 만들기
q 종료
ACTION MENU
a 세션 attach
k 세션 kill
r 이름 변경
b 뒤로 가기
// INTERNALS
EXECUTION FLOW
실행
→
setup_mouse
→
main loop
→
show_list
→
action_menu
→
attach / kill / rename

main()과 action_menu()는 모두 while true 루프로 구현되어 있어 탐색 깊이와 관계없이 호출 스택이 누적되지 않습니다.
setup_mouse()는 시작 시 ~/.tmux.conf에 set -g mouse on이 없으면 자동으로 추가합니다.
