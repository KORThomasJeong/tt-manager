세션 ▸ 윈도우 ▸ 패널 순서로 중첩됩니다.
세션을 detach해도 프로세스는 살아있어요.
Prefix Key — 모든 단축키의 시작
Ctrl + b

tmux의 모든 단축키는 Prefix 키로 시작합니다.
Ctrl+b 를 누른 후 손을 떼고, 이어서 다음 키를 누르세요.
아래 표에서 Ctrl+b 다음에 나오는 키가 실제 동작합니다.

01

세션 Session

tmux 새 세션 시작
tmux new -s 이름 이름 있는 세션 생성
tmux ls 세션 목록 보기
tmux attach -t 이름 세션 재접속
tmux kill-session -t 이름 세션 강제 종료
tmux rename-session -t 이름 세션 이름 변경
Ctrl+b › d
세션 분리 (Detach)
세션 유지하고 빠져나오기
Ctrl+b › $
세션 이름 변경
현재 세션 이름 수정
Ctrl+b › s
세션 목록 / 전환
인터랙티브 세션 트리 보기
02

윈도우 (탭) Window

Ctrl+b › c
새 윈도우 생성
create — 새 탭 열기
Ctrl+b › ,
윈도우 이름 변경
현재 탭에 이름 붙이기
Ctrl+b › n
다음 윈도우
next — 오른쪽 탭으로
Ctrl+b › p
이전 윈도우
previous — 왼쪽 탭으로
Ctrl+b › 0–9
번호로 바로 이동
숫자 키로 탭 전환
Ctrl+b › w
윈도우 목록 보기
인터랙티브 탭 전환
Ctrl+b › &
윈도우 닫기
현재 탭 종료 (확인 필요)
Ctrl+b › l
최근 윈도우로
last — 이전에 본 탭으로
03

패널 (화면 분할) Pane

Prefix + %  → 좌우 분할 (Vertical Split)
← LEFT
RIGHT →
Prefix + "  → 위아래 분할 (Horizontal Split)
↑ TOP
↓ BOTTOM
Ctrl+b › %
좌우로 분할
세로 선으로 화면을 나눔
Ctrl+b › "
위아래로 분할
가로 선으로 화면을 나눔
Ctrl+b › ↑↓←→
패널 간 이동
방향키로 인접 패널 전환
Ctrl+b › z
패널 전체화면 토글
zoom — 다시 누르면 복구
Ctrl+b › o
다음 패널로
순서대로 패널 순환
Ctrl+b › q
패널 번호 표시
번호 누르면 해당 패널로
Ctrl+b › { / }
패널 위치 교환
앞/뒤 패널과 자리 바꾸기
Ctrl+b › x
패널 닫기
현재 패널 종료
04

기타 Misc

Ctrl+b › ?
단축키 전체 목록
q 로 나가기
Ctrl+b › [
스크롤 모드
위아래 스크롤 · q 로 종료
Ctrl+b › :
명령어 직접 입력
tmux 명령 프롬프트 열기
Ctrl+b › t
시계 표시
큼직한 시각 표시 · q 닫기
